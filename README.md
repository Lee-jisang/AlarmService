# 딥러닝 모델을 이용한 기상알림 서비스
**얼굴객체 인식을 통한 알림 중지 기능 구현한 웹사이트**

## 1. 개발 환경 및 사용 라이브러리
- **OS** : Window 10
- **Language** : Python 3.8, HTML5, JavaScript, CSS
- **Framework** : Django
- **DBMS** : MySQL
- **Library** : Python Channels, Web Socket, OpenCV, imutils, dlib
